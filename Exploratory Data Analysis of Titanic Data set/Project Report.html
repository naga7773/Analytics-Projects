Details about the data set:¶

The titanic data set is taken from the kaggle website. This data set is a sample taken from the complete passenger list. There is no information regarding how this sample data is taken. I used a question and answer format to explore the dataset. some of the information about the dataset in the website is:

VARIABLE DESCRIPTIONS: survival Survival (0 = No; 1 = Yes) pclass Passenger Class (1 = 1st; 2 = 2nd; 3 = 3rd) name Name sex Sex age Age sibsp Number of Siblings/Spouses Aboard parch Number of Parents/Children Aboard ticket Ticket Number fare Passenger Fare cabin Cabin embarked Port of Embarkation (C = Cherbourg; Q = Queenstown; S = Southampton)

SPECIAL NOTES: Pclass is a proxy for socio-economic status (SES) 1st ~ Upper; 2nd ~ Middle; 3rd ~ Lower

Age is in Years; Fractional if Age less than One (1) If the Age is Estimated, it is in the form xx.5

With respect to the family relation variables (i.e. sibsp and parch) some relations were ignored. The following are the definitions used for sibsp and parch.

Sibling: Brother, Sister, Stepbrother, or Stepsister of Passenger Aboard Titanic Spouse: Husband or Wife of Passenger Aboard Titanic (Mistresses and Fiances Ignored) Parent: Mother or Father of Passenger Aboard Titanic Child: Son, Daughter, Stepson, or Stepdaughter of Passenger Aboard Titanic

Other family relatives excluded from this study include cousins, nephews/nieces, aunts/uncles, and in-laws. Some children travelled only with a nanny, therefore parch=0 for them. As well, some travelled with very close friends or neighbors in a village, however, the definitions do not support such relations.

Conclusions:¶

While the overall survival rate is 38.38%, female survival rate is 74.2% where as male survival rate is 18.89%. Of the 342 people who survived, 68.1% are females and 31.9% are males. It suggests that females had more chances of survival. When separated in to different age categories, kids survival rate is more (59.37%) compared to other categories with seniors having a survival rate of only 22.72%. This suggests that children had more chances of survival. There are almost double the number of passengers in 3 class compared to 2nd class. The survival rate for a 3 class passenger is 24.23% where as that of 2nd class passenger is 47.28% and 1st class passenger is 62.96%. These results clearly shows that kids and females had more chances of survival than other categories. Besides, 1st class passengers had more chances of survival. But, the extent to which this dependecy exists can only be explained by further statistical tests. There are many missing values in the age column. Deleting these rows may cause loss of valuable information. Instead of deleting theese rows, a guess value based on the other values in the adjacent columns would be helpful in further analysis. For example, if in the name colmn it is given as Master, then age can be guessed as a kid.¶
